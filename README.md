# Use this package :

## 1. Depend on it

Add this to your package's pubspec.yaml file:

```
dependencies:
  feedback_page: ^0.0.1
```
## 2. Install it

You can install packages from the command line:

with Flutter:

```
$ flutter pub get
```

## 3. Import it

Now in your Dart code, you can use:

```
import 'package:feedback_page/feedback_page.dart';
```
